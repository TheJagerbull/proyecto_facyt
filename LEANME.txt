El nombre que usara la Base de Datos es 'deca_admin'

En la Carpeta TablasBD se encuentran los scripts de SQL para la construccion o creacion de las tablas
de la base de datos, se les aconseja que si modificaron algo, exportenlo del phpmyadmin, y luego lo guardan con el nombre de la tabla
el archivo "relaciones de almacen.sql", contiene la identificacion de claves foraneas de todas las tablas de la BD, recuerden que si
se detecta algun cambio en estos archivos, deben notificarlo con el commit, y deben tambien actualizar regularmente
la base de datos local con estos scripts.

En la Carpeta DatosBD se encuentran las inserciones sobre las bases de datos, recuerden que si quieren agregar algo mas, deben hacerlo manualmente
sobre el script, sin previamente haber verificado que no este repetido el elemento a insertar.

Para aquellos que deseen importar datos desde una tabla de excell a un script de SQL, visiten el siguiente enlace
http://tools.perceptus.ca/text-wiz.php?ops=7
